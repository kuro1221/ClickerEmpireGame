# ClickerEmpireGame

Recursion Project3 ClickerEmpireGame

## Description
コンピュータ学習プラットフォームRecursionの課題の一つです。<br>
ハンバーガーをクリックしてお金を稼ぎ、資産を増やしていくゲームです。

サンプル画面

- Bootstrap
- Pure JavaScript

## Feature
- 画面遷移
- セーブ機能
